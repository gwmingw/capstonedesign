# 🤖 Prompt Logs: Unsupervised Event Abstraction

이 문서는 본 프로젝트의 핵심 파이프라인 코드를 AI 코딩 어시스턴트(Cursor, Claude Code 등)를 이용해 생성할 때 사용된 프롬프트 기록입니다. 동일한 파이프라인을 재현하거나 응용할 때 참고할 수 있습니다.

---

## 💬 Prompt

너는 프로세스 마이닝과 머신러닝에 정통한 시니어 데이터 사이언티스트야. 
아래의 8단계 지시사항에 따라, 센서(PLC) 로우 데이터를 비지도 학습 기법으로 추상화하고 프로세스 마이닝 맵을 도출하는 전체 파이프라인을 Python 코드로 작성해 줘. 각 단계별로 주석과 함께 결과를 확인할 수 있는 출력(print 또는 그래프) 코드를 포함해야 해.

### 🛠 [기술 스택]
`Python`, `numpy`, `pandas`, `scikit-learn`, `matplotlib`, `pm4py`

### 📝 [단계별 지시사항]

**1. 가상 PLC 데이터 생성**
- 길이 10의 바이너리 배열 패턴 4개(Phase 0~3)를 정의해 줘. (예: Phase 0은 `[1,1,0,0...]`, Phase 1은 `[0,0,1,1...]` 방식)
- 각 Phase가 10 스텝씩 유지되며 순환하도록 총 1,000 타임스텝의 시계열 데이터를 만들어 줘.
- 현실성을 위해 `np.random.rand`와 `np.logical_xor`를 사용하여 전체 데이터에 약 **2%의 무작위 노이즈**를 추가해 줘.

**2. 슬라이딩 윈도우 전처리**
- 시계열 맥락을 반영하기 위해 `window_size=5`로 슬라이딩 윈도우 함수를 구현하고 데이터를 변환해 줘. (결과 데이터 모양은 50차원이 되어야 함)

**3. PCA 차원 축소 및 시각화**
- `scikit-learn`의 PCA를 사용해 데이터를 3차원으로 축소해 줘.
- PC1, PC2, PC3의 개별 분산 설명력(`explained_variance_ratio_`)과 총 정보 유지율을 출력해 줘.
- 3차원 산점도(3D Scatter Plot)로 결과를 시각화해 줘.

**4. K-Means 클러스터링**
- `n_clusters=4`로 설정하여 K-Means 모델을 학습시키고 클러스터 라벨을 추출해 줘.
- 3단계에서 그린 3D 산점도에 클러스터 라벨로 색상을 입히고, Centroid를 빨간색 'X' 마커로 표시해 줘.

**5. 정량적 성능 평가**
- `silhouette_score`를 계산하여 클러스터링 품질을 소수점 4자리까지 출력해 줘.

**6. 시각적 평가 (시계열 변화)**
- x축을 타임스텝, y축을 클러스터 인덱스로 하여 공정이 어떻게 전환되는지 계단형 그래프(`drawstyle='steps-post'`)로 그려 줘.
- 전체 데이터가 너무 길 수 있으니 처음 10사이클(400 스텝) 정도만 잘라서 보여줘.

**7. 이벤트 로그 데이터프레임 생성**
- `pm4py` 입력을 위해 데이터를 변환해 줘.
- timestamp는 "2026-05-05 10:00:00"부터 1초 단위로 증가하도록 설정.
- case_id는 "Mitsubishi_Line_1"으로 고정.
- activity는 방금 구한 클러스터 라벨을 "Phase_0", "Phase_1" 형태로 변환해서 할당.

**8. 프로세스 맵(DFG) 도출**
- `pm4py.format_dataframe`을 사용해 포맷팅을 맞춰 줘.
- `discover_dfg`를 통해 Directly-Follows Graph를 도출하고, `view_dfg`를 사용하여 시각화하는 코드까지 작성해 줘.
